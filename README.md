# soltee-events-discord-automation-n8n

Automated customer enquiry system built with n8n for a UK decoration business.

## Tools Used
- n8n
- Webhooks
- Discord API

## What it does
When a customer submits an enquiry through the website contact form, their details (name, email, phone, service, event date, message) are instantly delivered to Discord via n8n automation.

## Why n8n
Built the same automation previously with Make.com, now rebuilt in n8n to demonstrate cross-platform automation skills and explore self-hosted options for client cost reduction.
